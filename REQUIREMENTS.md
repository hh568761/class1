
# 黄老师极客攻防实验室 - 教学大纲 (V3.0)

## 一、 核心教学宗旨
*   **功能驱动，视觉反馈**：每一课教一个真实的编程/安全概念（如端口扫描、协议握手、爬虫、加解密），绘图仅作为逻辑执行的直观结果。
*   **黑客语境**：所有的变量、注释和任务都设定在“渗透测试”和“系统安全”的语境下。
*   **真实交互**：引入 `fetchURL` 等函数，让学生能够真实抓取网页 HTML，而不仅是模拟。

## 二、 30 课完整大纲

### 第一阶段：终端渗透与系统基础 (1-8 课)
*   L1: 接入控制台 (cout 与黑客登录 Slogan)
*   L2: 系统权限鉴定 (if/else 模拟 sudo 校验)
*   L3: 端口扫描器 (for 循环遍历，雷达视觉)
*   L4: 内存缓冲区监控 (变量作用域与内存地址概念)
*   L5: 自动化指令流 (数组存储命令序列)
*   L6: 内核进程管理 (结构体定义 Process)
*   L7: 虚拟文件系统映射 (cd/ls 模拟，理解路径)
*   L8: 系统日志审计 (字符串过滤与搜索)

### 第二阶段：网络协议与数据侦听 (9-15 课)
*   L9: HTTP 三次握手 (握手协议逻辑仿真)
*   L10: 初探爬虫：HTML 抓取 (fetchURL 真实网页请求)
*   L11: 数据清洗提取器 (字符串截取新闻标题)
*   L12: 十六进制嗅探器 (数据转 Hex 显示)
*   L13: 子网掩码与位运算 (& 运算计算网络号)
*   L14: DNS 劫持模拟 (Map 结构映射域名)
*   L15: 流量洪峰：DoS 模拟 (多并发逻辑视觉化)

### 第三阶段：加密解密与安全算法 (16-23 课)
*   L16: 凯撒移位加密 (字符位移算法)
*   L17: 异或 (XOR) 混淆 (数据混淆与还原)
*   L18: 暴力破解 (Brute Force 耗时计算)
*   L19: 哈希碰撞初步 (Hash 函数冲突模拟)
*   L20: SQL 注入原理 (字符串拼接漏洞展示)
*   L21: 数字签名验证 (公钥私钥概念)
*   L22: 木马传播算法 (递归模拟网络扩散)
*   L23: 特征码扫描 (反病毒引擎基本原理)

### 第四阶段：实战演习与大师项目 (24-30 课)
*   L24: 自动化搜集脚本 (综合爬取外链)
*   L25: 黑客看板 (UI 综合集成)
*   L26: 模拟 Shell (命令行交互界面)
*   L27: 数据包分片重组 (网络传输底层仿真)
*   L28: 隐写术 (LSB 图片藏密)
*   L29: 自动防御系统 (IP 频率封禁逻辑)
*   L30: 毕业项目：夺旗赛 (综合运用)
